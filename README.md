**CrystalDash** is a stunning, futuristic admin dashboard template built around the popular **Glassmorphism** design trend. Featuring frosted glass cards, blurred floating orbs, and a built-in Light/Dark mode toggle, this template is perfect for SaaS platforms, analytics portals, and modern web applications.

## ✨ Key Features

* **3D Glassmorphism UI:** Translucent frosted glass cards (`glass-card-3d`) over an animated, multi-orb background.
* **Light & Dark Mode:** Integrated theme toggle functionality right in the navigation bar.
* **Modern Typography:** Beautifully paired 'Outfit' and 'Space Mono' Google Fonts.
* **Data Visualization Ready:** Pre-designed UI components for Revenue Analytics (bar charts), Traffic Sources (donut charts), and Project Progress bars.
* **Comprehensive Widgets:** Includes a dynamic monthly calendar widget, recent activity feeds, and a recent transactions data table.
* **Responsive Sidebar:** Fully collapsible mobile-friendly navigation menu.

## 🧩 UI Components Included

This dashboard is packed with ready-to-use structural elements:
* **Stats Grid:** Total Revenue, Active Users, Total Orders, and Conversion Rate cards with percentage indicators.
* **Activity Feed:** Timeline style list for tracking user actions.
* **Data Tables:** Clean, structured tables with user avatars and status badges (Completed, Processing, Pending).
* **Navigation:** Top navbar with global search and notification bells.

## 🚀 How to Use / Installation

Because this is a static HTML/CSS template, no complex backend or package manager is required.

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/yourusername/crystaldash-glassmorphism-admin.git](https://github.com/yourusername/crystaldash-glassmorphism-admin.git)
    ```
2.  **Open locally:** Simply double-click the `index.html` file to launch the dashboard in your browser.
3.  **Customize:** Connect the pre-built chart HTML/CSS to your favorite JavaScript charting library (like Chart.js or ApexCharts) to make the data dynamic.
